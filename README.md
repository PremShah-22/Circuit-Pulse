## 📖 Overview

**CircuitPulse** is an advanced interactive web application that simulates **Alternating Current (AC) circuits** in real time. Instead of simply displaying formulas and calculations, the project provides dynamic visualizations that help users understand how electrical parameters change as circuit components are adjusted.

Users can modify resistance, inductance, capacitance, voltage, and frequency while instantly observing changes in impedance, current, power, resonance, and phase angle.

The goal of this project is to make AC circuit analysis more intuitive through interactive graphics and live mathematical computation.

* ⚡ Real-time AC circuit calculations

## 🧮 Calculations Performed

The simulator computes:

* Resistance (R)
* Inductive Reactance (XL)
* Capacitive Reactance (XC)
* Net Reactance
* Total Impedance (Z)
* RMS Current
* Phase Angle
* Resonant Frequency
* Active Power (P)
* Reactive Power (Q)
* Apparent Power (S)
* Power Factor

All values update instantly whenever an input parameter changes.

Users can modify:

## 📊 Visualizations

### ⚡ Phasor Diagram

Displays electrical vectors including:

* Resistance Vector
* Inductive Reactance
* Capacitive Reactance
* Total Impedance

---

### 🌊 Oscilloscope

Shows animated sine waves of:

* Voltage
* Current

including the phase difference between them.

---

### 📉 Frequency Response

Displays how total impedance changes across different frequencies, helping users understand resonance and frequency-dependent behavior.
